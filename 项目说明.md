## 五、常用「调整旋钮」

### 倒计时目标时间
`script.js` 第一行：
```js
const WEDDING_DATE = new Date('2026-11-10T15:00:00+08:00'); // 宾客签到时间
```

### 首屏文字大小
`styles.css` 中：
- `.hero__names`（名字）`font-size: clamp(3.8rem, 13.5vw, 9.8rem)`
- `.hero__date`（日期）`clamp(1.6rem, 4.4vw, 2.6rem)`
- `.hero__place`（地点）`clamp(1.1rem, 2.8vw, 1.5rem)`

### 倒计时 / 邀请函在主图上的位置（仅桌面端 ≥761px）
`styles.css` 的 `@media(min-width:761px)` 里：
```css
.topshot > .countdown { top: 47%; }                 /* 上下位置 */
.topshot > .invitation { top: 85%; right: 5%; width: min(560px,48%); }  /* 上下/左右/宽度 */
```
- 往上 = 调小 `top%`，往下 = 调大 `top%`。
- 邀请函往左 = 调大 `right%`，往右 = 调小 `right%`。
- 容器高度 `.topshot { height: 133.4vw }` = 主图原始比例（高/宽 = 1.334），保证整张图完整显示。手机端（<761px）自动改为垂直堆叠。

### 酒店横幅 logo 大小
`styles.css` 的 `.hotel-hero__logo` → `width: min(680px, 92%)`

## 六、如何重新生成「白色透明底 logo」

如果更换了酒店 logo，需要重新生成白色版本（横幅用），在 `酒店信息/` 目录运行：

```bash
python3 -c "
from PIL import Image
img = Image.open('酒店logo.jpg').convert('RGB')
px = img.load(); w,h = img.size
out = Image.new('RGBA',(w,h),(0,0,0,0)); op = out.load()
for y in range(h):
    for x in range(w):
        r,g,b = px[x,y]
        lum = 0.299*r+0.587*g+0.114*b
        op[x,y] = (255,255,255,int(max(0,min(255,255-lum))))
out.save('酒店logo-white.png')
"
```
